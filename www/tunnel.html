No SSH tunnel open

You need to open a SSH tunnel to where the HTTP traffic will be proxied.
To do so just run the following command on you computer. Don't forget to replace $USER, $HOST and $PORT with the right values.
To get thoses value just run dotcloud info webtunnel.proxy and look for the ssh url ssh://$USER@$HOST:$PORT

Also replace localhost:8080 to point to your webserver. If you don't have any just run python -m SimpleHTTPServer to get one running on port 8080.

To open a tunnel

ssh -i ~/.dotcloud/dotcloud.key -l $USER -p $PORT $HOST -R 8042:localhost:8080
